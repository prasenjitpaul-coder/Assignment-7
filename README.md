This project is part of Assignment 7, where the objective is to identify and plan RESTful APIs required for a To-Do List application.
The focus is only on backend API planning, not full implementation.

The backend is structured using Node.js and Express.js, following proper MVC architecture (Routes, Controllers, Models).<br>
